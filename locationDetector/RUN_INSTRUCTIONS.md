## Start the Server (1 command)

```bash
./run_server.sh
```

## Open in Browser

**Computer:**
```
http://localhost:8000
```

**Phone (same WiFi):**
```
http://YOUR_IP:8000
```

Find your IP:
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

## That's It!

1. Click **"Start Live Stream"**
2. Allow camera access
3. Point at indoor environment
4. Watch metadata appear every 3 seconds!

## What You'll See

- Scene type (hallway, room, etc.)
- Room numbers and signage
- Landmarks (doors, stairs, exits)
- Directions and distances
- Quality indicators
